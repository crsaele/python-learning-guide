# 🐍 Guía de Especialización en Python
 
Una guía interactiva de una sola página pensada para quienes quieren aprender Python desde cero o estructurar mejor su camino. No es un curso, es una referencia visual clara que puedes consultar en cualquier momento.
 
## ¿Qué incluye?
 
- **Qué es Python** — definición, filosofía del lenguaje y ejemplo de código real
- **Para qué sirve** — las 6 grandes áreas: web, datos, IA, automatización, seguridad y cloud
- **Qué puedes construir** — proyectos concretos ordenados por nivel de experiencia
- **Ruta de aprendizaje** — 5 fases desde los fundamentos hasta nivel experto, con tiempos estimados
- **Cómo practicarlo** — plataformas recomendadas, el método de proyectos reales y la regla de los 20 minutos diarios
- **Cómo retener lo que aprendes** — técnicas basadas en neurociencia: repetición espaciada, Feynman, diario de aprendizaje y más
## Cómo usarlo
 
Abre `index.html` directamente en tu navegador. No requiere instalación, dependencias ni servidor — todo está contenido en un solo archivo.
 
## Tecnologías
 
HTML · CSS · JavaScript vanilla — sin frameworks, sin librerías externas.
